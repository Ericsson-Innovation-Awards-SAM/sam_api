#Sustainable Resource Mapping/Management (SAM) API

## Standards
- Python 3.4.5

## Installation
- `pip install virtualenv`
- `virtualenv -p python3 sam_venv`
- `source sam_venv/bin/activate`
- `pip install -r requirements.txt`

## Development
- `source sam_venv/bin/activate`

## Running the Application
- `python -m lib.main`
